brainf-ck
=========

###setup

````bash
$ cd brainf-ck
$ make
````

###run
````bash
$ ./bf -f helloworld.bf
Hello, world!
````
or
````bash
$ ./bf -e '+++++++++[>++++++++>+++++++++++>+++++<<<-]>.>++.+++++++..+++.>-.------------.<++++++++.--------.+++.------.--------.>+.>++[>+++++<-]>.'
Hello, world!
````
